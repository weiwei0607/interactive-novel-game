# Truth in the Mist (霧中真相) - Interactive Mystery Novel

An interactive mystery visual novel built with React, TypeScript, and Vite. 
Choose your character, explore scenes, gather clues, and find the culprit in this immersive detective experience.

## Demo Mode
To experience a quick preview without going through the entire gameplay loop, try the **Demo Mode**. This mode allows you to explore the core mechanics, UI, and early narrative paths instantly.

## Features
- Interactive dialogue system
- Scene exploration and clue gathering
- Responsive UI optimized for mobile devices
- Immersive storytelling and atmospheric design

## Development
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build for production: `npm run build`
